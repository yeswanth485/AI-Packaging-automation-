## 🔧 Common Tasks

### Access the Server
```bash
# Server is already running at http://localhost:3000

# To restart:
cd "d:\AI PACKAGING OPTIMIZER"
npm start

# For development with auto-reload:
npm run dev
```

### Setup Database
```bash
# 1. Install PostgreSQL or use Docker
# 2. Set DATABASE_URL in .env
# 3. Run migrations
npx prisma migrate deploy

# 4. Optional: Open Prisma Studio
npm run prisma:studio
```

### Build & Deploy
```bash
# Rebuild everything
npm run build:all

# Build just backend
npm run build

# Build just frontend
cd frontend && npm run build

# Start production server
npm run start:prod
```

### Docker Deployment
```bash
# Start all services
docker-compose up -d

# Status check
docker-compose ps

# Stop all services
docker-compose down
```
