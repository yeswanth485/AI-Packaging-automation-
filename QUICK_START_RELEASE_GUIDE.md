## 🎪 HOW TO USE

### For Demo/Testing
```
1. Open http://localhost:3000
2. Register demo account
3. Explore all features
4. Test optimization
5. Try batch upload
6. View analytics
```

### For Development
```
1. Backend: npm run dev
2. Frontend: cd frontend && npm run dev
3. Both accessible at localhost:3000
4. Hot reload enabled
5. Database: Set up PostgreSQL
6. Run migrations: npx prisma migrate deploy
```

### For Production
```
1. npm run build          - Build backend
2. cd frontend && npm run build  - Build frontend
3. npm run start:prod     - Start production
4. Or use Docker: docker-compose up
5. Set environment variables
6. Deploy to your platform
```
